# Java Arcade Game

Simple 2D Java game built using an educational drawing library `StdDraw` and data structures `tads`.

## Structure

- `Assets/` - images used by the game
- `base/` - base classes for shapes, graphical objects and the game loop
- `proyecto/` - game code

## Requirements

- Java 8 or newer
- `stdlib.jar` (included in the repository)
- `tads-20230510.jar` (included as well)

## Run

Compile and run the "main" method at `proyecto/Ejecutar.java`.
